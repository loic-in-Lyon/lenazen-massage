Projet Lenazen — Site vitrine
Contexte
Ce projet a été réalisé dans un cadre pédagogique par trois élèves développeurs web. Il s'agit de la création complète du site vitrine de la marque Lenazen, appartenant à Claire Cheviyer Gandon.

Objectifs
Concevoir un site moderne et responsive
Mettre en valeur les services de la marque
Travailler en collaboration sur un projet réel
Équipe
Développement / intégration / UI / UX design / Contenu par : Loïc Poo-sing – Khaoula Boudjema – Lucas Benarble
Statut du projet
Projet terminé — mis en ligne pour démonstration pédagogique et portfolio.
